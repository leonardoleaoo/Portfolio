# Portfólio | Leonardo Leão

Este é o meu **primeiro projeto liberado abertamente no GitHub**.

Sempre construí muita coisa para cliente, operação e produto, mas quase tudo ficava fechado em contexto de trabalho.  
Agora decidi publicar este portfólio para mostrar como eu penso interface, experiência e execução técnica de ponta a ponta.

## O que você vai encontrar aqui

Este site foi desenhado para ser direto, visualmente forte e com foco profissional:

- Hero com posicionamento claro e chamada para ação
- Seção de atuação técnica e visão de negócio
- Seção de stack principal com nível de domínio e foco atual
- Grid de projetos gerado por JavaScript
- Contatos rápidos (WhatsApp, Instagram e GitHub)
- Versão bilíngue (`PT/EN`) com persistência de idioma
- Animações com GSAP + ScrollTrigger

## Estilo e proposta visual

Escolhi uma linha **Neo-Brutalista Moderna**:

- Fundo branco
- Preto sólido em bordas e texto
- Roxo como cor de destaque
- Bordas marcadas e sombras duras
- Interações com efeito de “botão físico”

## Stack do projeto

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla)`
- `GSAP + ScrollTrigger` (CDN)
- `Tailwind CDN` (apoio utilitário)

## SEO e métricas

O projeto já está preparado com:

- Meta tags de SEO
- Open Graph e Twitter Card
- `schema.org` (`Person`)
- `favicon.svg`, `og-cover.svg` e `site.webmanifest`
- Estrutura pronta para `GA4` e `Plausible` no `script.js`
